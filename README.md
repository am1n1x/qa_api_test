# SimbirSoft SDET Project

## Описание

Этот проект представляет собой набор тестов для API, реализующих CRUD-операции с сущностями. Тесты написаны на Python с использованием `pytest` и интеграцией с Allure для генерации отчетов. Также используется Docker для запуска API-сервиса и базы данных.

## Стек технологий

- **Язык программирования**: Python 3.10
- **Тестовый фреймворк**: pytest
- **Отчеты**: Allure
- **Контейнеризация**: Docker, Docker Compose
- **База данных**: PostgreSQL
- **CI/CD**: GitHub Actions

## Структура проекта

- `api/requests/` — Клиенты для работы с API и модели запросов/ответов.
- `data/` — Генерация тестовых данных.
- `tests/` — Набор тестов для проверки API.
- `.github/workflows/run-tests.yml` — Конфигурация CI для автоматического запуска тестов.
- `requirements.txt` — Зависимости проекта.
- `.env` — Файл с переменными окружения.

## Особенности реализации
- Генерируются уникальные данные для каждого теста для потокобезопасного параллельного запуска.
- Для каждого теста заново создается запись в базе данных, что обеспечивает изоляцию и независимость тестов друг от друга. 
- Реализовано GitHub Actions для автоматического запуска тестов при каждом пуше в репозиторий, но есть ошибка: Github почему-то не может найти репозиторий с Allure.
